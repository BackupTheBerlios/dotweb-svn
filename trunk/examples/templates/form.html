DotWeb: Form Example

This example shows you some of DotWeb's form handling capabilities in action. When you play with this form a bit and take a closer look at the source code and the template of this example in the next step you will recognize that there is not a single line of PHP code that fills all values into the form after you have submitted it and there is also no PHP code to check if all fields are filled out.

You will only find that there are <dotweb:fieldvalidator> tags in the template which do all the validation stuff for you. So you only have to place some of these validation tags in your template and DotWeb will do the rest of the work for you.
